# Laporan Hasil Belajar Siswa

Sinopsis Aplikasi <br> <br>
Aplikasi ini adalah aplikasi yang akan membantu dan memudahkan seseorang dalam mencari Hasil Laporan Belajar’nya secara Online. Aplikasi yang sertai dengan penjelasan keberadaan sekolah, tentang sekolah, dll. <br>
Dalam aplikasi ini user akan di berikan beberapa menu yaitu “Beranda”, “Hasil Belajar Siswa”, “Tentang”,”Kontak”. <br>
Beranda yang  berarti menu utama user. <br>
Sekolah/Hasil Belajar Siswa adalah menu yang dimana user bisa melihat hasil Laporan Belajar Siswa dan tentang sekolah yang telah disediakan. <br> 
Tentang adalah User bisa mengetahui tentang aplikasi Hasil Belajar Siswa. <br>
Kontak adalah User bisa mengetahui info kontak Admin dan bisa memberi komentar.

Manfaat :
- Memudahkan seseorang dalam mengetahui Hasil Belajar’nya secara Online.
- Membuat orang menjadi tidak kesusahan jika kelupaan nilai’nya.
